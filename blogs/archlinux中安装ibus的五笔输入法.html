1. 先用pacman安装：

    2. ibus ibus-table cmake

    pacman -S ibus ibus-table cmake
  2. 下载:
    3. ibus-table-chinese-{当前版本}-Source-Standalone.tar.gz

    该包中包含MODULES,在执行后续的camke指令时，不会报错

  3. 解压
    4. tar xzvf ibus-table-chinese-{当前版本}-Source-Standalone.tar.gz
  4. 进入解压的文件夹DIR，能够看到tables目录，执行：
    5. cmake tables make 5.退回到上级目录，执行： cmake ibus-table-chinese make 6.安装table们,执行： sudo make install
      大功告成！ 执行cmake指令时，可能会找不到cmake的，cmake会提示下载相应的MODULE，下载完成解压到ibus-table-chinese的目录中。
