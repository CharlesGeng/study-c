1 print和import的更多信息

1.1 使用逗号输出

  1. 说明：使用print时，也可以在语句中添加多个表达式，每个表达式用逗 号分隔；
  2. 注意：在用逗号分隔输出时，print语句会在每个输出项后面自动添加一 个空格；
  3. 例子： 
     1:  >>> greeting = 'Hello'
 2:  >>> salution = 'Mr.'
 3:  >>> name = 'Bill'
 4:  #以逗号分隔输出项
 5:  >>> print(greeting, salution, name)
 6:  Hello Mr. Bill
 7:  #在逗号前增加了一个空格符
 8:  >>> print(greeting, ',', salution, name)
 9:  Hello , Mr. Bill
10:  #为了显示成'Hello, Mr. Bill'这个样式，可以使用连接符‘+’
11:  >>> print(greeting + ',', salution, name)
12:  Hello, Mr. Bill
13:  >>>
14:  
15:

1.2 把一些东东作为另一些东东导入

  1. 说明：
    1. 将整个模块导入，格式为：import somemodule;
    2. 从某个模块中导入某个函数,格式为：from somemodule import somefunction;
    3. 从某个模块中导入多个函数,格式为：from somemodule import firstfunc, secondfunc, thirdfunc
    4. 将某个模块中的全部函数导入，格式为：from somemodule import *
  2. 注意：如果两个模块中都有相同的函数，则可以使用第一种方法导入模块， 也可以使用关键字 as 为相同的函数取个别名,例子： 
     1:  #第一种导入方法
 2:  #导入模块
 3:  import module1
 4:  import module2
 5:  #调用同名函数的方法
 6:  module1.open()
 7:  module2.open()
 8:  
 9:  #第二种导入方法
10:  #导入函数，并给函数取相应的别名
11:  from module1 import open as open1
12:  from module2 import open as open2
13:  
14:
  3. 例子： 
     1:  #从math中导入sqrt
 2:  >>> from math import sqrt as msqrt
 3:  #从cmath中导入sqrt
 4:  >>> from cmath import sqrt as csqrt
 5:  >>> msqrt(100)
 6:  10.0
 7:  >>> csqrt(-1)
 8:  1j
 9:  >>>
10:

2 赋值魔法

15:  'second'

  2. 例子：

2.3 增量赋值

  1. 说明：包括以下增量操作：
    1. +=:将右侧的值加到变量上的和，然后再赋值给变量；
    2. -=:将变量减去右侧的值得到的差，再赋值给变量；
    3. /=:用变量除以右侧值得到的商，再赋值给变量；
    4. %=:用变量取右侧值的余数，再赋值给变量；
  2. 注意： += 和 \*= 还可以应用在字符串上，见下面的示例；
  3. 例子： 
     1:  #针对数字的各种操作
 2:  >>> x = 123
 3:  >>> x += 1
 4:  >>> x
 5:  124
 6:  >>> x -= 4
 7:  >>> x
 8:  120
 9:  >>> x *=2
10:  >>> x
11:  240
12:  >>> x /=3
13:  >>> x
14:  80.0
15:  >>> x %=9
16:  >>> x
17:  8.0
18:  >>>
19:  
20:  #字符串的增量赋值
21:  >>> y = 'Test string'
22:  >>> y += ', haha!'
23:  >>> y
24:  'Test string, haha!'
25:  >>> y *= 2
26:  >>> y
27:  'Test string, haha!Test string, haha!'
28:  >>>

4 条件和条件语句

4.1 这就是布尔变量的作用

  1. 说明：布尔值，
    1. 假值：false，None，所有类型的数字0，空序列，空字典；
    2. 真值：所有的非空值；
    3. bool函数可以用来将其他值转换成布尔值；
  2. 注意：尽管假值具有不同的类型，但是不同的假值之前也是 不相等 的
  3. 例子： 
     1:  >>> True
 2:  True
 3:  >>> False
 4:  False
 5:  >>> []
 6:  []
 7:  >>> bool ([])
 8:  False
 9:  >>> bool ([1,])
10:  True
11:  >>> bool (0)
12:  False
13:  >>> bool (0.0)
14:  False
15:  >>> bool (0.1)
16:  True
17:  #不同的假值之间也是不相同的
18:  >>> [] == {}
19:  False
20:  >>> [] == None
21:  False
22:  >>>

4.2 条件执行和if语句

  1. 说明：if 判断其后面的条件语句是否为真，如果为真，执行if后面的语句 块，否则不执行；

4.3 else子句

  1. 说明：之所以称为子句是因为else必须跟在if语句后面，而不能单独使用；

4.4 elif子句

  1. 说明：如果需要更多的判断，可以使用elif，判断更多的条件；
  2. 例子： 
     1:  #if, elif, else应用
 2:  num = input("Please enter a number:")
 3:  num = int(num)
 4:  if num > 0:
 5:          print ('You input a positive number!')
 6:  elif num < 0:
 7:          print ('You input a negative number!')
 8:  else:
 9:      print ('You input a zero!')

4.6 更复杂的条件

4.6.1 比较运算符

  1. 说明：
    1. x==y:　等于；
    2. x<y: 　小于；
    3. x>y: 大于；
    4. x<=y:　小于等于；
    5. x>=y:　大于等于；
    6. x!=y:　不等于；
    7. x is y:x和y是同一对象；
    8. x is not y:x和y不是同一对象；
    9. x in y:　x在y中；
    10. x not in y:　x不在y中；
  2. 注意：
    1. 比较运算符是可连接的，例如：14 < age < 26；
    2. 比较运算符不能比较不同类型的数据；

4.6.2 相等运算符

  1. 说明：用来判断两个数据是否相等；

4.6.3 同一性运算符

  1. 说明：用于判断两个变量是否指向同一对象；
  2. 注意：避免把 is 比较运算符应用于比较常量值，如数字，字符串等。 即 避免 以下比较: 
    1:  if '123' is '123':
2:

4.6.4 成员资格运算符

  1. 说明：判断元素是否被包含在对象中；

4.6.5 字符串和序列比较

  1. 说明：字符串可以按照字母顺序排列进行比较；

4.6.6 布尔运算符

  1. 说明：包括,and, or, not
  2. 例子：

 2:  >>> name = input("Please enter a name:") or '<unknown>'

11:  'a'
12:  #如果if后面的判断语句为假，返回c
13:  >>> a if False else c
14:  'c'

5.4 一些迭代工具

5.4.1 并行迭代

  1. 说明：zip内置函数可以将多个序列“压缩”成一个元组的序列；
  2. 例子：

  1. 说明：内建函数reversed用于翻转序列，内建函数sorted用于对序列排 序，他们都是返回操作后的序列，不对原序列进行修改；
