1 除法

1.1 除号－/

  1. 说明： / 是除法符号，两个整型数相除，如果商不是整数，会显示小数。
  2. 注意：商虽然是小数，但是最后一位不会做四舍五入运算。
  3. 例子： 
1:  >>> 1/3
2:  0.3333333333333333
3:  >>> 2/3
4:  0.6666666666666666
5:

1.2 除号－//

  1. 说明：//是整除符号，两个浮点数相除，商也会只保留整数。
  2. 注意：
    • 整除 不 会进行四舍五入操作；
    • 如果除数和被除数都是整型，则结果也是整型；
    • 如果除数和被除数有其中一个是浮点数，则结果也是浮点数。
  3. 例子： 
1:  >>> 1 // 3
2:  0
3:  >>> 2 // 3
4:  0
5:  >>> 2.9 // 3
6:  0.0
7:  >>> 3.1 // 3
8:  1.0

2 幂运算符

  1. 说明：** 用于计算乘方。
  2. 注意：
    1. 幂运算比取反运算优先级高；
  3. 例子： 
1:  >>> 2 ** 3
2:  8
3:  >>> 3 ** 2
4:  9
5:  >>> -3 ** 2
6:  -9
7:  >>> (-3)** 2
8:  9

10 模块

  1. 说明：可以把模块想像成功能集。用 import 关键字导入模块
    1. 导入模块的方法1：import 模块
    2. 导入模块的方法2：from 模块 import 函数
  2. 注意：应该多使用第一种模块导入的方法，这样就不会因为不同模块中有相 同函数而导致冲突。
  3. 例子： 
 1:  #python 3.2中，math.floor直接返回整型，与int的效果相同
 2:  #使用第一种模块导入方法
 3:  >>> import math
 4:  >>> math.floor(32.99999)
 5:  32
 6:  >>> int(32.99999)
 7:  32
 8:  
 9:  #使用第二种模块导入方法
10:  >>> from math import sqrt
11:  >>> sqrt(169)
12:  13.0
13:  
14:  #cmath模块
15:  >>> import cmath
16:  >>> cmath.sqrt(-1)
17:  1j

11 字符串

11.1 单引号和双引号

  1. 说明：打印时，单引号和双引号都可以使用。
  2. 注意：在字符串中有单引号时，可以使用双引号，在字符串中有双引号时， 可以使用单引号
  3. 例子 
1:  >>> print ("This is a test!")
2:  This is a test!
3:  >>> print ('I said:"I will be there!"')
4:  I said:"I will be there!"
5:  >>> print ("Let's go!")
6:  Let's go!
7:

11.2 转义字符

  1. 说明：对字符串中的引号用 \ 进行转义
  2. 例子： 
1:  >>> print ("\"")
2:  "
3:  >>> print ('\'')
4:  '
5:

11.3 拼接字符串

  1. 说明：用加法符号拼接 + 字符串
  2. 例子： 
1:  >>> print ("Let's say:\"" + "Hello, world!\"")
2:  Let's say:"Hello, world!"
3:

11.4 值被转换为字符串的两种机制

  1. 说明：两种机制
    • 用str转换，它会把字符串转换成合法形式的字符串，以便用户可以理 解。
    • 用repr转换，它会创建一个字符串，以合法的Python表达式的形式来表 示值。 
 1:  #str与repr的区别
 2:  >>> c = r"c:\nowhere"
 3:  >>> c
 4:  'c:\\nowhere'
 5:  >>> print(c)
 6:  c:\nowhere
 7:  >>> print(str(c))
 8:  c:\nowhere
 9:  >>> print(repr(c))
10:  'c:\\nowhere'
11:

1:  >>> name = input("Your name: ")

3:  >>> print("Hello " + name)

11.6 长字符串

  1. 说明：
    1. 用三个双引号来引用长字符串．
    2. 在字符串的行尾添加 \
  2. 注意：三个双引号的字符串会保留换行符，而反斜线不会保留，见示例
  3. 例子： 
 1:  #三个引号的示例
 2:  
 3:  >>> print ("""
 4:  ... This is a very long string.
 5:  ... It continues here.
 6:  ... END.
 7:  ... """)
 8:  
 9:  This is a very long string.
10:  It continues here.
11:  END.
12:  
13:  #反斜线的示例
14:  >>> print ("This is a long string \
15:  ... It continues here.\
16:  ... This is the end!")
17:  This is a long string It continues here. This is the end
18:

11.7 原始字符串

  1. 说明：在字符串前面添加字母 r ， 这在正则表达式中非常有用！
  2. 注意：
    1. 原始字符串不能以反斜线结尾；
    2. 如果需要在原始字符串输出时以反斜线结尾，可以按最后一个例子的 方法实现。
  3. 例子： 
 1:  #反斜线被当成转义字符打印出来
 2:  >>> print ("C:\nowhere")
 3:  C:
 4:  owhere
 5:  
 6:  #直接打印反斜线
 7:  >>> print (r"C:\nowhere")
 8:  C:\nowher
 9:  
10:  #不能在原始字符串结尾使用反斜线
11:  >>> c = r"test\"
12:  SyntaxError: EOL while scanning string literal
13:  
14:  #在原始字符串结尾输出反斜线的方法
15:  >>> print (r"C:\windows""\\")
16:  C:\windows\
17:

11.8 Unicode字符串

  1. 说明：在字符串前加字母 u 来表示立unicode字符串，但是这个功能已 经在python3中废弃了。
  2. 例子： 
1:  >>> u'中文'
2:  SyntaxError: invalid syntax
